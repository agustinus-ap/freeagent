= FreeAgent Client

(Unofficial) Ruby client for the {Freeagent Central API}[http://www.freeagentcentral.com/developers/freeagent-api].


== Requirements

* Ruby >= 1.8.7
* ActiveResource 3.0.x


== Installation

  $ gem install freeagent
